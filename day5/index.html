    Day 5：整理一份旅行计划清单¶

    前四天我们学了变量、判断和循环。今天来学两个非常重要的数据结构：列表和字典。它们是 Python 里最常用的"容器"，用来把一堆数据整理在一起。

    场景¶

    你打算和朋友去旅行，需要整理一份清单：去哪些地方、每个地方玩几天、预算多少。这些信息零零散散的，用单个变量装不下。

    列表和字典就是干这个的：帮你把一组相关的数据收纳在一起。

    列表（list）：有序的数据集合¶

    列表就是一组数据排成一排，用方括号 [] 包起来，元素之间用逗号隔开：

    cities = ["成都", "大理", "厦门", "西安"]

    如果类比 Excel，列表就像一列单元格，从上到下排列。

    访问列表元素¶

    列表里的每个元素都有一个编号（叫做"索引"），从 0 开始数：

    cities = ["成都", "大理", "厦门", "西安"]

print(cities[0])    # 成都（第一个）
print(cities[1])    # 大理（第二个）
print(cities[-1])   # 西安（最后一个，-1 是倒着数）

    修改、添加和删除¶

    cities = ["成都", "大理", "厦门", "西安"]

# 修改某个元素
cities[2] = "杭州"
print(cities)    # ['成都', '大理', '杭州', '西安']

# 在末尾添加
cities.append("重庆")
print(cities)    # ['成都', '大理', '杭州', '西安', '重庆']

# 删除指定元素
cities.remove("大理")
print(cities)    # ['成都', '杭州', '西安', '重庆']

# 查看列表长度
print(len(cities))    # 4

    用 for 循环遍历列表¶

    Day 4 学的 for 循环和列表是天生一对：

    cities = ["成都", "大理", "厦门", "西安"]

for city in cities:
    print(f"{city}，我来了!")

    如果想同时拿到编号和内容，用 enumerate()：

    cities = ["成都", "大理", "厦门", "西安"]

for i, city in enumerate(cities, 1):
    print(f"第 {i} 站：{city}")

    输出：

    第 1 站：成都
第 2 站：大理
第 3 站：厦门
第 4 站：西安

    enumerate(cities, 1) 里的 1 表示编号从 1 开始数。不写的话默认从 0 开始。

    列表的其他常用操作¶

    numbers = [3, 1, 4, 1, 5, 9, 2, 6]

# 排序
numbers.sort()
print(numbers)    # [1, 1, 2, 3, 4, 5, 6, 9]

# 反转
numbers.reverse()
print(numbers)    # [9, 6, 5, 4, 3, 2, 1, 1]

# 判断某个值是否在列表里
print(5 in numbers)     # True
print(7 in numbers)     # False

# 求和、最大值、最小值
print(sum(numbers))     # 31
print(max(numbers))     # 9
print(min(numbers))     # 1

    字典（dict）：有标签的数据集合¶

    列表用编号来定位数据，字典用名字来定位。用花括号 {} 包起来，每个元素是一个"键: 值"对：

    trip = {
    "目的地": "大理",
    "天数": 5,
    "预算": 3000,
    "已订机票": True
}

    如果列表像 Excel 的一列，那字典就像一行：每个单元格都有列标题。

    访问和修改¶

    trip = {
    "目的地": "大理",
    "天数": 5,
    "预算": 3000,
}

# 通过键名获取值
print(trip["目的地"])    # 大理
print(trip["预算"])      # 3000

# 修改
trip["预算"] = 3500
print(trip["预算"])      # 3500

# 添加新的键值对
trip["同行人数"] = 3
print(trip)

# 删除
del trip["同行人数"]

    遍历字典¶

    trip = {"目的地": "大理", "天数": 5, "预算": 3000}

# 遍历所有键值对
for key, value in trip.items():
    print(f"{key}：{value}")

    输出：

    目的地：大理
天数：5
预算：3000

    嵌套：列表和字典的组合¶

    真正强大的地方在于列表和字典可以互相嵌套。比如一份旅行计划，包含多个目的地，每个目的地有自己的详细信息：

    travel_plan = [
    {"城市": "成都", "天数": 3, "预算": 2000, "必去": "锦里、宽窄巷子"},
    {"城市": "大理", "天数": 4, "预算": 2500, "必去": "洱海、古城"},
    {"城市": "厦门", "天数": 3, "预算": 2200, "必去": "鼓浪屿、曾厝垵"},
]

    这是一个列表，列表里的每个元素是一个字典。类比 Excel 的话，整个列表是一张表，每个字典是一行，键名是列标题。

    处理这种嵌套数据：

    print("=== 旅行计划 ===")
print()

total_days = 0
total_budget = 0

for stop in travel_plan:
    print(f"{stop['城市']}：{stop['天数']}天，预算{stop['预算']}元")
    print(f"  必去：{stop['必去']}")
    total_days += stop["天数"]
    total_budget += stop["预算"]

print()
print(f"总行程：{total_days} 天")
print(f"总预算：{total_budget} 元")

    列表推导式：一行代码生成列表¶

    Python 有一个很酷的语法，可以用一行代码从已有数据生成新列表：

    numbers = [1, 2, 3, 4, 5]

# 每个数字乘以 2
doubled = [n * 2 for n in numbers]
print(doubled)    # [2, 4, 6, 8, 10]

# 只保留偶数
evens = [n for n in numbers if n % 2 == 0]
print(evens)      # [2, 4]

    一个实用的例子：从旅行计划里提取所有城市名：

    city_names = [stop["城市"] for stop in travel_plan]
print(city_names)    # ['成都', '大理', '厦门']

    列表推导式刚接触时可能会觉得有点绕，不用着急，先学会基本写法，后面用多了就自然了。

    小练习¶

    练习 1：购物清单管理器

    写一个程序，让用户不断输入想买的东西和价格，输入 "done" 结束。然后显示完整清单和总价。

    提示：用一个列表存储字典，每个字典包含"名称"和"价格"两个键。

    练习 2：通讯录

    创建一个列表，里面存几个人的信息（姓名、电话、邮箱）。然后让用户输入一个名字，搜索并显示对应的联系方式。如果没找到，提示"查无此人"。

    练习 3：成绩排名

    用一个列表存储几个同学的名字和成绩（列表嵌套字典），按成绩从高到低排序后输出排名。

    提示：sorted() 函数可以对列表排序，sorted(students, key=lambda x: x["成绩"], reverse=True) 可以按字典中某个键来排。这个语法看起来复杂，先照着用，后面会理解的。

    今日小结¶

    今天你学会了：

    • 列表：有序数据集合，用 [] 创建，用索引访问
    • 字典：带标签的数据集合，用 {} 创建，用键名访问
    • 列表和字典可以嵌套组合，表示复杂的数据结构
    • 列表推导式：一行代码从已有数据生成新列表
    • 类比 Excel：列表像一列，字典像一行，列表嵌套字典就像一张表

    明天我们要学"函数"，把重复的代码逻辑封装起来，让程序更有条理。
