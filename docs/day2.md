# Day 2：帮朋友算 AA 制聚餐费用

昨天我们认识了 Python，今天来做一件实际有用的事：写一个 AA 制聚餐费用计算器。

## 场景

周末朋友们一起吃了顿火锅，结账时要 AA。掏出手机算来算去，还得考虑有人没喝酒少付点。与其费这个劲，不如让 Python 帮你算。

要做这件事，我们需要学会三样东西：**变量**、**数据类型**和**输入输出**。

## 变量：给数据贴个标签

你在 Excel 里填数据时，每个单元格都有自己的位置（比如 A1、B2）。Python 里做类似的事情，用的是**变量**。

变量就是一个名字，指向一个数据。你给它起个名，然后把数据"装"进去：

```python
total = 580
people = 4
```

这两行代码做了什么事？

- 创建一个叫 `total` 的变量，里面存的是 580（聚餐总金额）
- 创建一个叫 `people` 的变量，里面存的是 4（一共 4 个人）

之后你随时可以用这个名字来取出数据：

```python
print(total)
print(people)
```

变量的值可以随时修改：

```python
total = 580
print(total)    # 输出 580

total = 620
print(total)    # 输出 620，新值覆盖了旧值
```

### 变量命名规则

变量名不能随便起，有几个简单的规则：

- 只能用英文字母、数字和下划线 `_`，不能用中文（技术上可以，但别这么干）
- 不能以数字开头。`price1` 可以，`1price` 不行
- 大小写敏感。`name` 和 `Name` 是两个不同的变量
- 建议用有意义的名字。`total` 比 `x` 好懂得多

## 数据类型：数据也分种类

Python 里的数据分几种类型，就像 Excel 里的单元格格式有"数字"、"文本"、"日期"一样。现在你只需要知道四种：

**字符串（str）** -- 文本内容，用引号包起来：

```python
name = "小明"
restaurant = '海底捞'
```

单引号和双引号都行，效果一样。

**整数（int）** -- 没有小数点的数字：

```python
people = 4
age = 25
```

**浮点数（float）** -- 有小数点的数字：

```python
total = 580.50
tip_rate = 0.1
```

**布尔值（bool）** -- 只有两个值，`True`（真）或 `False`（假）：

```python
is_weekend = True
has_discount = False
```

布尔值在 Day 3 学条件判断时会大量用到，今天先知道它的存在就行。

想看一个变量是什么类型，可以用 `type()`：

```python
print(type(580))       # <class 'int'>
print(type(580.5))     # <class 'float'>
print(type("你好"))    # <class 'str'>
print(type(True))      # <class 'bool'>
```

## 让程序和用户对话：input 和 print

`print()` 你已经用过了，用来输出信息。`input()` 则是让程序等用户输入内容：

```python
name = input("你叫什么名字？")
print("你好，" + name + "!")
```

运行这段代码后，程序会在终端里显示"你叫什么名字？"，然后停下来等你输入。你输入一个名字按回车后，它会打印出问候语。

### 一个重要的坑：input 拿到的永远是字符串

不管用户输入什么，`input()` 返回的都是字符串类型。即使用户输入了一个数字，Python 也把它当成文字来看待：

```python
number = input("请输入一个数字：")
print(type(number))    # <class 'str'>，不是 int!
```

如果你想拿它来做计算，需要手动转换类型：

```python
number = input("请输入一个数字：")
number = int(number)       # 把字符串转成整数
print(number * 2)          # 现在可以做数学运算了
```

常用的类型转换：

- `int("123")` -- 把字符串转成整数
- `float("3.14")` -- 把字符串转成浮点数
- `str(100)` -- 把数字转成字符串

## 做计算

Python 支持这些数学运算：

```python
print(10 + 3)     # 加法，结果 13
print(10 - 3)     # 减法，结果 7
print(10 * 3)     # 乘法，结果 30
print(10 / 3)     # 除法，结果 3.3333...（总是返回浮点数）
print(10 // 3)    # 整除，结果 3（只取整数部分）
print(10 % 3)     # 取余，结果 1（除不尽剩下的部分）
print(10 ** 3)    # 乘方，结果 1000（10 的 3 次方）
```

四舍五入用 `round()`：

```python
print(round(3.14159, 2))    # 保留 2 位小数，结果 3.14
print(round(145.5))         # 不指定位数，四舍五入到整数，结果 146
```

## 格式化输出：f-string

到目前为止，我们用加号 `+` 来拼接字符串。但如果要拼接数字和文字，还得手动转类型，写起来很啰嗦。

Python 有一个更优雅的方式叫 **f-string**。在字符串的引号前面加一个 `f`，然后用花括号 `{}` 包裹变量名：

```python
name = "小明"
age = 25
print(f"{name}今年{age}岁")    # 输出：小明今年25岁
```

花括号里还可以直接写表达式：

```python
price = 580
people = 4
print(f"每人应付{price / people}元")    # 输出：每人应付145.0元
print(f"每人应付{price / people:.2f}元")  # 输出：每人应付145.00元
```

`:.2f` 的意思是保留两位小数。处理金额时经常用到。

## 动手做：AA 制计算器

学了这些，我们来写一个完整的 AA 制聚餐费用计算器：

```python
print("=== AA 制聚餐费用计算器 ===")

total = float(input("请输入聚餐总金额："))
people = int(input("请输入聚餐人数："))

per_person = total / people

print(f"总金额：{total:.2f} 元")
print(f"聚餐人数：{people} 人")
print(f"每人应付：{per_person:.2f} 元")
```

保存成 `.py` 文件运行，试试看效果。

这段代码的逻辑：
1. 用 `input()` 让用户输入总金额和人数
2. `float()` 和 `int()` 把输入转成数字
3. 算出每人应付的金额
4. 用 f-string 格式化输出结果
