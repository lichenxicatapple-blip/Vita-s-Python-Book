# 八天学会 Python -- 写给零基础文科生的编程入门教程

你好！欢迎来到这份 Python 入门教程。

这份教程专门写给**完全没有编程经验**的朋友。不需要你懂任何技术背景，只要会用电脑打字、上网，就能跟着学下来。

我们用 8 天时间，从零开始，一步步学会 Python 的基础用法。到最后一天结束时，你将能用 Python 写出一个真正有用的数据分析小工具。

## 每天学什么

| 天数 | 主题 | 你将学会 |
|------|------|----------|
| Day 1 | 和 Python 打个招呼 | 安装 Python 和编辑器，写出第一行代码，感受编程的乐趣 |
| Day 2 | 帮朋友算 AA 制聚餐费用 | 用变量存数据，做计算，让程序和用户对话 |
| Day 3 | 做个小测验分类游戏 | 让程序学会判断，根据不同情况做不同的事 |
| Day 4 | 批量处理不求人 | 让程序自动重复劳动，一次处理一批数据 |
| Day 5 | 整理一份旅行计划清单 | 用列表和字典管理一组数据，就像整理表格 |
| Day 6 | 别重复自己 | 用函数封装重复的逻辑，让代码更有条理 |
| Day 7 | 认识对象 | 理解 Python 里"万物皆对象"的设计思想 |
| Day 8 | 你的第一个数据分析小工具 | 安装第三方库、读写 Excel 文件、综合数据分析项目 |

## 环境准备

Day 1 会手把手带你完成 Python 和编辑器的安装，这里先做个简要说明，让你心里有数。

我们需要装两样东西：

1. **Python** -- 我们要学的编程语言本身。去 https://www.python.org/downloads/ 下载最新版。
2. **VS Code** -- 一个免费的代码编辑器，用来写代码和运行程序。去 https://code.visualstudio.com/ 下载。

具体的安装步骤、每一步该点哪里，都在 Day 1 的教程里写得清清楚楚，到时候照着做就行。

## 学习建议

- **每天 30-60 分钟就够了。** 贪多嚼不烂，坚持比冲刺重要。
- **一定要动手敲代码。** 看懂和会写是两回事。每个示例都亲手敲一遍，别复制粘贴。
- **别怕报错。** 报错是 Python 在告诉你哪里不对，不是在骂你。学会读错误信息是编程进步最快的方式。
- **多试多改。** 看完示例后，试着改改参数、换换写法，看看会发生什么。
- **遇到困难很正常。** 每个程序员都是这么过来的。卡住了就休息一下，明天再看可能就豁然开朗了。
