    Day 1：和 Python 打个招呼¶

    今天是第一天，我们的目标很简单：把 Python 装好，写出你人生中的第一行代码。

    为什么要学 Python¶

    你可能每天都在用 Excel 处理数据：复制粘贴、拖公式、做透视表。这些操作重复又枯燥，稍不留神还容易出错。

    Python 能帮你把这些重复劳动自动化。比如：

    • 自动读取 10 个 Excel 文件，汇总成一张表
    • 批量重命名几百个文件
    • 从一堆数据里筛选出你要的内容，生成报告

    Python 是目前最流行的编程语言之一，语法简洁，读起来接近英语，对新手非常友好。

    什么是编程¶

    编程，说白了就是写一份指令清单，让电脑按你说的去做事。

    就像你写一份菜谱：先洗菜、再切菜、然后下锅炒。电脑不会自己思考，但它严格执行你写的每一条指令，而且速度极快、从不喊累。

    Python 就是我们用来写这份"指令清单"的语言。

    第一步：安装 Python¶

    1. 打开浏览器，访问 https://www.python.org/downloads/
    2. 页面上有一个醒目的黄色按钮，写着 "Download Python 3.x.x"，点击它
    3. 下载完成后，在访达（Finder）的"下载"文件夹里找到刚下载的 .pkg 文件，双击打开
    4. 按提示一路点击"继续"、"同意"、"安装"
    5. 系统会要求输入你的电脑密码（就是开机时输入的那个密码），输入后点"安装软件"
    6. 等待安装完成，看到"安装成功"就好了

    第二步：认识终端¶

    在学写代码之前，我们先认识一个重要的工具：终端（Terminal）。

    终端是 Mac 自带的一个应用程序。它的界面很朴素，就是一个可以输入文字命令的窗口。你输入一行命令，电脑就执行一行。我们之后会用终端来运行 Python 程序。

    打开终端¶

    按 Command+空格（键盘左下角那个花瓣形状的键 + 空格键），会弹出一个搜索框（这个搜索框叫 Spotlight）。输入"终端"（或者输入 "Terminal"），然后按回车。

    你会看到一个窗口打开了，里面有一行文字和一个闪烁的光标，等你输入命令。这就是终端。

    验证 Python 是否安装成功¶

    在终端里输入下面这行命令，然后按回车：

    python3 --version

    如果看到类似 Python 3.13.2 这样的输出，说明 Python 安装成功了。

    在 Mac 上我们用 python3 而不是 python 来运行 Python。这是因为 Mac 系统历史上自带过一个旧版本，用 python3 能确保运行的是我们刚装的新版本。

    理解 cd 命令：在文件夹之间移动¶

    你平时用访达（Finder）的时候，想打开桌面上的某个文件，会双击桌面文件夹，找到文件，再双击打开。在终端里做同样的事情，靠的是 cd 命令。

    终端在任何时候都"站在"某个文件夹里。它只能看到当前文件夹里的文件，就像你在访达里打开了一个文件夹，只能看到这个文件夹里的东西一样。

    cd 是 "change directory" 的缩写，意思是"切换目录"（目录就是文件夹，两个词是一回事）。用法很简单：

    cd 你想去的文件夹路径

    举几个例子：

    # 去桌面
cd ~/Desktop

# 去"下载"文件夹
cd ~/Downloads

# 去"文稿"文件夹
cd ~/Documents

    这里的 ~ 是一个特殊符号，代表你的用户主目录（就是访达侧边栏里你名字对应的那个文件夹）。~/Desktop 就是"我的主目录下面的 Desktop 文件夹"，也就是你的桌面。

    想看看自己当前在哪个文件夹里，输入 pwd 按回车，终端会告诉你。

    想看看当前文件夹里有什么文件，输入 ls 按回车，终端会列出来。

    不用死记硬背这些命令，用几次就自然记住了。后面的教程里每次用到都会提醒你。

    第三步：安装 VS Code（代码编辑器）¶

    写代码需要一个专门的编辑器。我们用 VS Code，它免费、好用，是目前最流行的代码编辑器。

    下载和安装¶

    1. 打开浏览器，访问 https://code.visualstudio.com/
    2. 页面上有一个蓝色的下载按钮，点击它（网站会自动识别你是 Mac 用户）
    3. 下载完成后，在"下载"文件夹里找到一个 .zip 文件，双击解压
    4. 解压后会出现一个叫 "Visual Studio Code" 的应用程序图标
    5. 把它拖到访达左侧边栏的"应用程序"文件夹里
    6. 然后从"应用程序"里双击打开它。第一次打开时 Mac 可能会弹窗问"确定要打开吗"，点"打开"就行

    安装 Python 扩展¶

    打开 VS Code 后，我们需要装一个小插件，让 VS Code 更好地支持 Python。

    1. 看 VS Code 左侧边栏，有一列小图标。找到从上往下数第五个图标，长得像四个小方块拼在一起。把鼠标放上去会显示 "Extensions"。点击它
    2. 顶部会出现一个搜索框，输入 Python
    3. 搜索结果里第一个应该是 "Python"，下面写着 Microsoft。点击它右侧的蓝色 Install 按钮
    4. 等几秒钟就装好了。装好后这个按钮会变成一个小齿轮图标

    VS Code 里也有终端¶

    VS Code 有一个很方便的功能：它内置了一个终端。这样你写代码和运行代码可以在同一个窗口里完成，不用来回切换。

    打开方式：点击 VS Code 顶部菜单栏的 Terminal -> New Terminal。

    VS Code 底部会出现一个区域，里面有闪烁的光标。这个内置终端和刚才打开的 Mac 终端用法一模一样，cd、ls、python3 这些命令都能用。

    以后写代码时推荐直接用 VS Code 的内置终端，省得在多个窗口之间跳来跳去。

    第一个程序：你好，Python¶

    终于到了激动人心的时刻！

    1. 在 VS Code 菜单栏点击 File -> New File
    2. VS Code 会在中间区域弹出一个选择列表，选择 Python File。如果没看到这个选项，就随便选一个，之后保存时改成 .py 结尾就行
    3. 在新打开的空白文件里输入这行代码：
    print("你好，Python!")
    1. 按 Command+S 保存。VS Code 会问你保存到哪里，选桌面就行，文件名叫 hello.py

    Python 文件的后缀名必须是 .py，就像 Word 文件是 .docx、Excel 文件是 .xlsx 一样。

    1. 打开 VS Code 的内置终端（Terminal -> New Terminal），先用 cd 切换到桌面，再运行程序：
    cd ~/Desktop
python3 hello.py

    你应该会看到屏幕上出现了：

    你好，Python!

    恭喜！这是你写的第一个程序。

    print() 是 Python 的一个内置功能，作用是把括号里的内容显示到屏幕上。括号里的文字要用引号包起来（单引号 ' 或双引号 " 都行）。

    再试试多写几行，加到 hello.py 里，保存后重新运行 python3 hello.py：

    print("我正在学 Python!")
print("1 + 1 =", 1 + 1)
print("Python 真好玩")

    试试 Python 的交互模式¶

    除了写文件运行，Python 还有一个"交互模式"：你输入一行代码，它立刻给你结果，适合用来做实验。

    在终端里输入 python3，按回车。你会看到类似这样的内容：

    Python 3.13.2 (main, ...)
Type "help", "copyright", "credits" or "license" for more information.
>>>

    >>> 就是 Python 在等你输入。试试：

    >>> 1 + 1
2
>>> "你好" * 3
'你好你好你好'
>>> 100 / 7
14.285714285714286

    Python 就像一个超级计算器，你说什么它算什么。注意第二个例子：一段文字乘以 3，Python 会把它重复三遍，是不是挺有意思的？

    想退出交互模式，输入 exit() 按回车就行。

    别怕报错¶

    新手最怕看到报错信息，满屏红字看着吓人。但报错只是 Python 在告诉你"这里有个问题"，不是在骂你。

    我们故意写错试试。在交互模式里输入：

    >>> prnt("你好")

    Python 会说：

    NameError: name 'prnt' is not defined

    翻译：prnt 这个名字我不认识。因为正确的写法是 print，少打了一个 i。

    再试一个：

    >>> print("你好)

    Python 会说：

    SyntaxError: unterminated string literal

    翻译：引号没有配对。因为我们漏了右边的引号 "。

    报错信息会告诉你两件事：错误的类型（比如 NameError、SyntaxError）和大致原因。养成阅读报错信息的习惯，这是编程进步最快的方式。

    小贴士：你在网上搜 Python 教程时，可能偶尔会看到 print "你好" 这种不带括号的写法。那是 Python 2 的老语法，已经过时了。我们用的是 Python 3，print 后面一定要加括号。

    小练习¶

    动手试试，在交互模式或新建 .py 文件里完成都行：

    练习 1：自我介绍

    用 print() 输出三行内容：你的名字、你的城市、你最喜欢的一本书。

    练习 2：Python 计算器

    用 Python 计算以下结果：

    • 365 乘以 24（一年有多少小时？）
    • 1000 除以 7（保留小数没关系）
    • 2 的 10 次方（提示：Python 里用 ** 表示乘方，写成 2 ** 10）

    练习 3：和报错做朋友

    故意写几行错误的代码，看看 Python 会给你什么样的报错信息。试着理解每条报错在说什么。比如：

    • 把 print 拼错
    • 漏掉一个引号
    • 漏掉一个括号

    今日小结¶

    今天你完成了这些事情：

    • 安装了 Python 和 VS Code
    • 认识了终端和 cd 命令，学会了在终端里输入命令
    • 用 print() 输出了你的第一行代码
    • 体验了 Python 的交互模式
    • 认识了报错信息，知道它们并不可怕
