    Vita's Python Book¶

    从零开始，八天时间，一步步学会 Python 的基础用法。

    课程目录¶

    DAY 1 和 Python 打个招呼 安装 Python 和编辑器，写出第一行代码，感受编程的乐趣 DAY 2 帮朋友算 AA 制聚餐费用 用变量存数据，做计算，让程序和用户对话 DAY 3 做个小测验分类游戏 让程序学会判断，根据不同情况做不同的事 DAY 4 批量处理不求人 让程序自动重复劳动，一次处理一批数据 DAY 5 整理一份旅行计划清单 用列表和字典管理一组数据，就像整理表格 DAY 6 函数：封装重复逻辑 用函数封装重复的逻辑，让代码更有条理 DAY 7 认识对象 理解 Python 里"万物皆对象"的设计思想 DAY 8 你的第一个数据分析小工具 安装第三方库、读写 Excel 文件、综合数据分析项目

    学习建议¶

    • 每天 30-60 分钟就够了。 贪多嚼不烂，坚持比冲刺重要。
    • 一定要动手敲代码。 看懂和会写是两回事。每个示例都亲手敲一遍，别复制粘贴。
    • 别怕报错。 报错是 Python 在告诉你哪里不对，不是在骂你。
    • 多试多改。 看完示例后，试着改改参数、换换写法，看看会发生什么。
    • 遇到困难很正常。 每个程序员都是这么过来的。卡住了就休息一下，明天再看可能就豁然开朗了。
